## KeenKeeper

KeenKeeper is a simple app to help you keep track of your friends and stay in touch regularly. You can log calls, texts, and video chats, and the app shows whether you’re on track or need to follow up.

### Tech Stack
- Tailwind CSS  
- React Router  
- React Icons  
- JavaScript  

### Features
- Track calls, texts, and video interactions  
- Shows status like On Track, Almost Due, Overdue  
- Clean and responsive design for all devices
